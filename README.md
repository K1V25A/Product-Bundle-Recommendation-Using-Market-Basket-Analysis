 ## Product Bundle Recommendation Using Market Basket Analysis and Time-Series Clustering
 
 Overview 

This project recommends product bundles for e-commerce businesses by analyzing customer purchase history. It combines Market Basket Analysis and Time-Series Clustering to identify products that are frequently bought together and understand their demand patterns.

The goal is to help businesses increase revenue, improve customer experience, and support better merchandising decisions.


 Objectives

- Analyze customer purchasing behavior
- Discover frequently purchased product combinations
- Generate association rules using the Apriori algorithm
- Group products based on demand trends using Time-Series Clustering
- Recommend high-value product bundles

 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- NetworkX


Techniques Used

- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Market Basket Analysis (Apriori Algorithm)
- Association Rule Mining
- Time-Series Clustering (K-Means)
- Bundle Opportunity Scoring

 Dataset

- Simulated e-commerce transaction dataset
- 12 months of sales data
- 38 products across 6 categories

 Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Market Basket Analysis
5. Time-Series Clustering
6. Bundle Recommendation
7. Business Insights

 Results

The project identifies:

- Frequently bought together products
- High-confidence association rules
- Demand trends for products
- Recommended product bundles for maximizing revenue
